# Projeto:
#### Simular um sistema de uma plataforma de cursos, simulando cadastro de alunos, cursos e matrículas. 

## Objetivo:
#### Criar uma API Rest integrada a um banco SQL com ORM utilizando o padrão MVC.

## Start project:
npm run dev

## Dependencies
- express
- sequelize
- sequelize-cli
- sqlite3

## DevDependencies:
- eslint
- nodemon
